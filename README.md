# 🚀 CryptoNews AI — Live Crypto News & Airdrops

> AI-powered cryptocurrency news platform with automatic content generation, live prices, and crypto airdrop listings.

🌐 **Live Site:** [https://ailivecrypto.vercel.app](https://ailivecrypto.vercel.app)

## ✨ Features
- 📰 Auto Articles — AI generates fresh crypto news every 4 hours
- 🪂 Crypto Airdrops — Auto-generated airdrop listings
- 💰 Live Prices — Real-time crypto prices via CoinGecko
- 🤖 AI Chat — Built-in crypto assistant powered by Groq
- 📊 Fear & Greed Index — Live market sentiment
- 📈 TradingView Charts — Live BTC/ETH charts

## 🛠️ Tech Stack
- Frontend: Next.js 14, TypeScript, Tailwind CSS
- Database: Supabase (PostgreSQL)
- AI: Groq LLaMA 3.3 70B
- Hosting: Vercel
- Automation: GitHub Actions

## 🪂 Airdrop System
Auto-generated every 4 hours via GitHub Actions.

## 📊 Pages
- / Home with live ticker and AI chat
- /articles All crypto news
- /prices Live top 50 crypto prices
- /airdrops Crypto airdrop listings

## 🔌 API
- POST /api/generate — Generate article
- POST /api/generate-airdrop — Generate airdrop
- GET /api/airdrops — List airdrops
